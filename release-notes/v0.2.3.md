# Winsentials v0.2.3 ✨

Released after `v0.2.2`, this update focuses on simplifying the app's appearance model and making core tweak workflows cleaner.

## Highlights 🚀

- Reworked appearance settings around a cleaner `light` and `dark` theme model while keeping `Acrylic`, `Mica`, and `Tabbed` materials.
- Added a dedicated `Tools` hub page with home-style cards instead of the previous sidebar dropdown flow.
- Expanded the `Input` section with a proper `Disable Mouse Acceleration` tweak for 1:1 cursor movement.
- Refined tweak cards, switch states, reset actions, and toast presentation across the app.

## New Tweaks 🛠️

- `Disable Mouse Acceleration`
  Disables Windows pointer acceleration for more predictable cursor movement in games and precision work.

## Improvements 🎨

- Simplified theming by removing the old palette system and consolidating the app around one modern light theme and one modern dark theme.
- Preserved desktop materials as modifiers, so visual effects still work independently from the selected theme.
- Rebuilt tweak card layout to better handle switches, selects, reset actions, and compact metadata.
- Replaced the `Tools` sidebar dropdown with a dedicated page for faster navigation between utility sections.
- Updated localized strings and category descriptions to reflect the expanded input tooling.
