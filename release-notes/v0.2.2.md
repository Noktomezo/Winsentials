# Winsentials v0.2.2 ✨

Released after `v0.2.1`, this update focuses on expanding the tweak library and tightening the desktop UI.

## Highlights 🚀

- Added new tweak coverage across Behaviour, Input, Network, Performance, and Security.
- Introduced dedicated `Input` and `Performance` sections in the sidebar.
- Expanded appearance controls with a proper window material selector for `None`, `Acrylic`, `Mica`, and `Tabbed`.
- Refined tweak cards, page headers, toasts, and overall visual polish across the app.

## New Tweaks 🛠️

- `Disable Startup App Delay`
  Lets startup apps launch immediately after sign-in instead of waiting for Explorer's default delay.
- `Fast Keyboard Repeat`
  Adds a dedicated input tweak for more aggressive key repeat behavior.
- `Disable NCSI Active Probing`
  Turns off Windows internet connectivity probing for users who want fewer background network checks.
- `Disable Fault Tolerant Heap`
  Disables Windows crash-mitigation compatibility behavior for apps that do not need it.
- `Disable GameDVR and Xbox Game Bar`
  Turns off Game Bar capture hooks and related background recording behavior.
- `Optimize MMCSS`
  Adds a performance-focused multimedia scheduling profile.
- `Disable AutoPlay`
  Disables automatic media and device playback prompts.

## Improvements 🎨

- Added window material selection to settings and startup bootstrap handling.
- Improved tweak metadata presentation, including richer prompts, badges, and card layout polish.
- Refined desktop window behavior and related Tauri-side window commands.
- Updated routing and navigation to surface the new tweak categories.
- Refreshed localization strings in both English and Russian for the expanded tweak set.
